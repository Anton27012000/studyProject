# Mental Math Trainer

Консольный тренажер устного счета на Go.

Проект тренирует:

- работу с функциями
- пакеты и структуру проекта
- циклы и switch
- валидацию ввода
- struct и методы
- работу со случайными числами

---

# Возможности

- случайные математические операции
    - сложение
    - вычитание
    - умножение
    - деление
- проверка ответа пользователя
- подсчет статистики
- защита от неверного ввода
- бесконечный игровой цикл
- выход через меню

---

# Архитектура

## main

Только запускает программу и управляет сценарием.

## handlers

Содержит игровые сценарии.

Например:

- запуск раунда
- проверка ответа
- обновление статистики

## mathgame

Вся математическая логика:

- генерация чисел
- случайная операция
- вычисление ответа

## input

Безопасный ввод данных из консоли.

## menu

Отображение интерфейса программы.

## stats

Хранение статистики:

- правильные ответы
- неправильные ответы

---

Из корня проекта:

```bash
go run ./mental-math-trainer
```
