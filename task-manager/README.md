# Console Task Manager

Консольный менеджер задач на Go.

Проект тренирует:

- Слайсы и range
- Строки, byte, rune и пакет strings
- Maps: быстрый поиск и подсчет
- struct и методы
- первые тесты

---

# Возможности

- случайные математические операции
    - сложение
    - вычитание
    - умножение
    - деление
- проверка ответа пользователя
- подсчет статистики
- защита от неверного ввода
- бесконечный игровой цикл
- выход через меню

---

# Архитектура

## main

Только запускает программу и управляет сценарием.

## handlers

Содержит игровые сценарии.

Например:

- запуск раунда
- проверка ответа
- обновление статистики

## mathgame

Вся математическая логика:

- генерация чисел
- случайная операция
- вычисление ответа

## input

Безопасный ввод данных из консоли.

## menu

Отображение интерфейса программы.

## stats

Хранение статистики:

- правильные ответы
- неправильные ответы
