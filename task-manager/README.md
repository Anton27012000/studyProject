# Console Task Manager

Консольный менеджер задач на Go.

---

## Проект тренирует:

- работу со структурами (`struct`)
- методы и pointer receiver
- слайсы (`[]Task`)
- поиск, фильтрацию и удаление элементов
- работу с `map` для статистики
- декомпозицию кода на пакеты
- работу со строками (`strings`)
- `TrimSpace`, `Contains`, `ToLower`
- различие между `byte` и `rune`
- нормализацию текста (Unicode)
- обработку пользовательского ввода
- защиту от неверных данных
- unit-тесты (`go test`)

---

# Возможности

- добавление задачи
- просмотр всех задач
- просмотр активных задач
- поиск задачи по ID
- поиск по части названия
- отметка задачи как выполненной
- удаление задачи
- статистика по задачам:
    - всего задач
    - выполненные
    - активные
    - по приоритетам (low / medium / high)
- нормализация ввода (TrimSpace + Unicode обработка)
- защита от пустого ввода и некорректных ID
- бесконечный цикл меню
- выход через меню

---

# Архитектура

## main

Только запускает программу и управляет сценарием.

- читает выбор пользователя
- вызывает handlers
- не содержит бизнес-логики

---

## handlers

Слой взаимодействия между пользователем и бизнес-логикой.

Например:

- добавление задачи
- вывод списка задач
- завершение задачи
- удаление задачи
- поиск задач
- вывод статистики

---

## storage

Основная бизнес-логика и работа с задачами:

- хранение `[]Task`
- генерация ID
- поиск по ID
- удаление задач
- фильтрация (активные задачи)
- поиск по названию
- обновление статуса
- вычисление статистики (`map[string]int`)

---

## models

Описание сущностей проекта:

- `Task`:
    - ID
    - Title
    - Priority
    - Done

---

## utils

Вспомогательные функции для работы со строками:

- `NormalizeTitle`
    - TrimSpace
    - ToLower
    - корректная обработка кириллицы через `rune`

---

## input

Безопасный ввод данных из консоли:

- чтение строк
- чтение чисел
- обработка ошибок ввода
- защита от падений программы

---

## menu

Вывод пользовательского меню:

- список действий
- форматирование интерфейса

---

## tests

Unit-тесты:

- добавление задачи
- поиск по ID
- завершение задачи
- статистика
- нормализация строки

---
